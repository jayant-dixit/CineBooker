About CineBooker

Movie Magic - Your Smart Ticket Booking System

With the increasing demand for a seamless and modern movie-watching experience, traditional ticket booking methods often fall short due to long queues, limited availability, and inconsistent service. To address this, we introduced CineBooker—a smart, cloud-based movie ticket booking system.

Built using Flask for backend development, hosted on AWS EC2, and integrated with DynamoDB for dynamic data management, our platform allows users to register, log in, and book movie tickets online with ease. Users can search for movies and events based on location, view real-time seat availability, and complete their bookings in just a few clicks.

Upon booking, AWS SNS sends instant email notifications confirming ticket details, enhancing user engagement and trust. This cloud-native solution streamlines the entire movie ticketing process, ensuring fast, scalable, and user-friendly access to entertainment for all.

Key Features:

  • ✅ User-friendly registration and login system
  • ✅ Location-based movie and event search
  • ✅ Real-time seat availability
  • ✅ Interactive seat selection
  • ✅ Instant booking confirmations
  • ✅ Email notifications via AWS SNS
  • ✅ Secure and scalable cloud infrastructure
